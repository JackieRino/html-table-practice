# html-table-practice
A simple HTML table created as part of a freeCodeCamp workshop to practice table structure and elements.

##Purpose
The goal of this project was to practice:
-Creating table using HTML
-Using table elements such as 'table','thead','tbody','tr','th' and 'td'
-Structuring tabular data correctly

##What I Learned
-How to build and sturcture HTML table
-The difference between table headers and table data
-How to preview changes locally using Visual Stusio Cose and a browser

##Technologies Used
HTML5
